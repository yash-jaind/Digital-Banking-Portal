# Digital Banking Portal

A web-based Digital Banking Portal developed using Java, JSP, Servlets, JDBC, and MySQL following the MVC architecture pattern. The application enables secure banking operations including account management, fund transfers, transaction tracking, and user authentication.

## Features

* Secure user authentication and session management
* Account creation and management
* Fund transfer between accounts
* Transaction history and tracking
* JDBC-based database connectivity
* MVC architecture for better code organization
* Relational database design using MySQL

## Technologies Used

* Java
* JSP
* Servlets
* JDBC
* MySQL
* HTML
* CSS
